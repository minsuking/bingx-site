## 1. 공식 사이트 접속
👉 [bingx.com/invite/YOURCODE](https://bingx.com/invite/YOURCODE)

## 2. 이메일 또는 휴대폰 등록
- 이메일, 비밀번호 입력 → 인증코드 확인  
- **레퍼럴 코드 자동 적용** 확인 후 가입 완료

## 3. 로그인 후 2단계 인증(2FA)
- Google Authenticator 설정
